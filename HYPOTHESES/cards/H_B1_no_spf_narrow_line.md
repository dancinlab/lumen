# H_B1 — no-SPF narrow-line recovery

> **🜂 ABSTRACT.** Brainstorm shortlist (`state/resist-laser-brainstorm.md`, C7). Unverified.

## Which wall
Flux (system-level in-band recovery) — recovers delivered watts without a brighter source.

## Principle
Broadband LPP needs spectral-purity filters (SPFs) that throw away ~25–30% of in-band power to
block out-of-band/IR. An intrinsically narrow source (recombination laser, H_B4) needs no SPF, so
that power reaches the wafer; the same narrow coherent line also serves as an actinic
mask-inspection source, amortizing cost.

## Falsifiable prediction
Dropping SPFs with a narrow (Δλ/λ < 10⁻³) source recovers ≥ 25% delivered in-band power vs a
broadband LPP at equal raw source power. If SPF-free out-of-band contamination forces equivalent
losses elsewhere, falsified.

## How it would be tested
Optical power-budget audit (SPF transmission × out-of-band rejection) for narrow vs broadband —
runnable next (a candidate to promote, like H_012). Not run here.

Unverified (🜂) but lowest-risk of the frontier set — depends on H_B4 (a narrow source existing).
